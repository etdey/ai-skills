# AI Skills

This is a collection of my skills for AI tools.

## Installation

The generalized installation instructions are to copy the entire skill
subdirectories into the location where your AI coding tool looks for
them. 

- GitHub Copilot CLI: `$HOME/.copilot/skills/`
- OpenCode: `$HOME/.config/opencode/skills/`

See each skill for any specific instructions for modifying the skill
based on your system configuration.
