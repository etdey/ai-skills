# Skill to Create Obsidian Note

Creates a new markdown note in an Obsidian vault from a chat session
conversation.

## Installation

See [project README](../README.md) for instructions the install this
skill for various AI shells.

Post-installation configuration steps:
- Edit the _Known configuration_ section of [SKILL.md](./SKILL.md)
- Install dependencies:
    - QuickJS
    - Node.js


## Automated Tests

Run the unit tests from this directory:
```sh
npm test
```
